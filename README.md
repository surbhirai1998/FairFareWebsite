# TaxiFarePredictionWebsite
ASP.NET Website for predicting fare of taxis using machine learning 

Using this website users can enter the :
1. Passenger Count
2. Trip time
3. Trip distance
4. Payment Type

Website Gives the cab fare that will be required for the travel.
